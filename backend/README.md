# latent

To install dependencies:

```bash
bun install
```

To run (development):

```bash
bun run dev
```

To run (production):

```bash
bun run start
```

### Database

To push schema changes:

```bash
bun run db:push
```

To open Prisma Studio:

```bash
bun run db:studio
```

This project was created using `bun init` in bun v1.3.5. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
